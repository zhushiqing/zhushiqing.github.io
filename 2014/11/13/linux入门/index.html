Linux入门

Linux分为内核版与发行版，发行版是基于内核版上进行二次开发从而得出的版本

进程 unix→minix→linux

redhat：centOS

fedaro

linux一般作为稳定的服务器，图形界面会占用系统资源，字符界面最好

linux严格区分大小写

linux所有都以文件形式存储，一切都是文件

linux常用命令：

  • 命令的基本格式：命令 [选项] [参数]

  • 文件处理的命令：

    • 目录处理命令：

      • 创建目录 mddir -p xxx 递归创建 如果不加-p就一个目录
      • cd cd .. cd . cd ~ cd - 上一次访问
      • rmdir : 删除空目录 rm -rf
      • cp -r 复制目录 cp复制文件 -p 连带文件属性复制 -d 若源文件是链接文件，则复制链接属性， -a 相当于-pdr
      • mv 源文件或者目录 剪切 也是改名 源文件跟目标文件在同一目录下
